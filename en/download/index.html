Hazelcast Jet Downloads

There are two different Hazelcast Jet download packages - regular and slim. The regular Hazelcast Jet download package includes Hazelcast Jet server and all production ready modules. The slim package includes only Hazelcast Jet server and all modules need to be downloaded separately (see Modules section below). Both require a JDK to run, which can be obtained from AdoptOpenJDK (minimum version is 8 - recommended is 11 or later). For details about what's included, and minimum requirements please see the installation page.
